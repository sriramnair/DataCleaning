Setup

Download the starter files on Frodo for this exercise. You will receive two files, artwork_data.csv and artwork_sample.csv.

Then, create a new file of your own called clean.py that will manipulate the data in the .csv files you downloaded.

Make sure you have successully installed pandas in your python environment. You can either install pandas globally on your machine (pip3 install pandas) or create a virtual environment for this exercise and install pandas in the environment.

$ virtualenv env
$ source env/bin/activate
$ (env) pip3 install pandas
